Nova Programming Language

A modern, dynamically-typed language built for clarity and simplicity. Implemented in C, batteries included.

git clone <repo-url> && cd nova && cmake -B build && cmake --build build

Easy to Learn

Clean syntax with only 22 keywords. No semicolons, no braces — just do and done. If you can read English, you can read Nova.

Batteries Included

Ships with 12 standard library modules covering math, file I/O, strings, JSON, OS, random, collections, paths, datetime, CSV, regex, and HTTP.

Extensible

Import modules, load C shared libraries, or install community packages from GitHub. Nova grows with your needs.

Fast

Implemented in C for speed. Nova's lightweight runtime starts instantly and runs your programs without unnecessary overhead.

A Quick Look

# Define a greeter class
class Greeter do
    func init(name) do
        self.name = name
    done

    func hello() do
        return "Hello, {self.name}!"
    done
done

let greeter = Greeter("World")
print(greeter.hello())
# Error handling with try/catch
import "io"
import "json"

try do
    let data = io.read_file("config.json")
    let config = json.parse(data)
    print("Loaded {len(config)} settings")
catch err do
    print("Could not load config: {err}")
done
